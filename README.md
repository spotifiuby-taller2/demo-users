# users

### Instalación y configuración

- #### Con Docker Compose
```
docker-compose up --build
```

- #### Sin Docker
    * `npm ci`
    * Como pre-requisito se debe tener una base de postgres slq que escuche en el puerto 5432
    * `npm run startdev`
